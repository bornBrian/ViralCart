## ğŸ› Common Issues & Solutions

### Issue: Products not showing
**Check:**
- Database connection (Supabase credentials)
- Run `database/seed.sql` for test data
- Browser console for errors

### Issue: Admin can't login
**Check:**
- URL has `?token=YOUR_TOKEN`
- Token matches environment variable
- No typos in token

### Issue: Click tracking not working
**Check:**
- Serverless function deployed
- Environment variables set on hosting
- Function logs for errors
- Browser console for CORS errors

### Issue: Slow loading
**Check:**
- Image sizes (compress them!)
- Lighthouse audit results
- CDN enabled (Netlify/Vercel auto-enable)
- Too many products? Add pagination
